# HAQAGen Project Website

This folder contains a **static** project website for:

**Histogram Assisted Quality Aware Generative Model for Resolution Invariant NIR Image Colorization** (Accepted to WACV 2026)

## Quick start (GitHub Pages)

1. Create a GitHub repo, for example `HAQAGen`.
2. Copy the contents of this folder into the repo root.
3. Commit and push.
4. In GitHub: **Settings â†’ Pages**
   - Source: `Deploy from a branch`
   - Branch: `main` (or `master`) and `/ (root)`

Your site will appear at:
`https://<your-username>.github.io/<repo-name>/`

## What to edit

- `index.html`
  - Update the "Code" and "Video" links (currently placeholders).
  - If you have a poster, demo, or additional links, add buttons in the hero section.
- `assets/paper.pdf`
  - Replace if you want a camera-ready or arXiv version.
- `assets/*.png`
  - Figures are cropped from the paper for web display.

## Notes

- Math is rendered with KaTeX via a CDN.
- No build tools are required.
